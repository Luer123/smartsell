# SmartSell 客服 WordPress 插件

一个功能强大的 WordPress 插件，用于在您的网站上接入具有主动销售能力的 AI 智能客服系统。

## ✨ 功能特性

### 核心功能

- 🤖 **AI 智能客服** - 集成先进的 AI 助手
- 💬 **会话管理** - 实时查看和管理所有客户会话记录
- 🎯 **线索追踪** - 自动识别和追踪潜在客户线索
- 👥 **客户管理** - 完整的客户信息管理和跟进系统
- 🏷️ **标签管理** - 灵活的标签系统，便于客户分类和管理
- 📚 **知识库同步** - 一键同步 WordPress 文章和 WooCommerce 商品到 AI 知识库
- ⚙️ **机器人配置** - 自定义机器人外观、名称、欢迎语等设置
- 📊 **数据统计** - 实时查看会话、线索、客户等关键数据

## 🔧 系统要求

- **WordPress**: 5.0 或更高版本
- **PHP**: 7.2 或更高版本
- **可选**: WooCommerce（用于商品同步功能）

## 📦 安装方法

### 方法一：通过 WordPress 后台安装

1. 登录 WordPress 后台
2. 进入 `插件` → `安装插件`
3. 点击 `上传插件`
4. 选择插件 ZIP 文件并上传
5. 点击 `现在安装`
6. 安装完成后点击 `启用插件`

### 方法二：手动安装

1. 下载插件文件
2. 将插件文件夹上传到 `/wp-content/plugins/` 目录
3. 在 WordPress 后台的 `插件` 页面中启用插件

## 🚀 使用指南

### 首次使用

1. **登录账户**
   - 安装并启用插件后，进入 `SmartSell客服` 菜单
   - 输入您的用户名和密码进行登录
   - 登录成功后，系统会自动获取您的应用信息

2. **配置机器人**
   - 进入 `机器人配置` 页面
   - 设置机器人头像、名称和欢迎语
   - 保存设置后，聊天机器人将自动显示在网站前端

3. **同步知识库**
   - 进入 `文章同步` 页面，选择要同步的文章
   - 进入 `商品同步` 页面（需要 WooCommerce），选择要同步的商品
   - 点击同步按钮，将内容同步到 AI 知识库

### 主要功能页面

- **会话管理** - 查看和管理所有客户会话
- **线索追踪** - 查看和管理潜在客户线索
- **线索跟进日志** - 查看线索的跟进历史
- **客户管理** - 管理客户信息和资料
- **客户跟进日志** - 查看客户的跟进记录
- **标签管理** - 创建和管理客户标签
- **文章同步** - 同步 WordPress 文章到知识库
- **商品同步** - 同步 WooCommerce 商品到知识库
- **机器人配置** - 配置聊天机器人的外观和行为
- **个人中心** - 查看个人信息和账户设置
